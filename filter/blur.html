filter : blur

The blur filter is not yet suitable for tilable patterns because it does not wrap around the edges. Copied from http://blog.ivank.net/fastest-gaussian-blur.html.

Basic Example

"patina" : {
            "type"          : "filter",
            "patternName"   : "noise_plasma",
            "filter"        : [{ "name": "blur" }]
        }

Options / Parameters

The blur filter takes only one parameter:

radius
possible values are positive numbers:
3 (default)
1
10
