filter : brightness

The brightness filter can be positive to brighten the color channel or negative to darken it. Values outside of -10 and 10 do not make much sense.

Example

                    
"patina": {
    "type"          : "createPattern",
    "patternName"   : "slope",
    "filter"        : [
        { 
            "name": "brightness", 
            "brightness": .25 
        }
    ]
}
