createPattern : wave

The wave patterns are calculated using the sine function. That gives us smooth lines in different directions.

Example

"patina" : {
    "type"          : "createPattern",
    "patternName"   : "wave"
}

Options / Parameters

direction
possible values are these strings:
vertical (default)
horizontal
diagonalUp
diagonalDown
concentric
rectangles
frequency
defines how many 'waves' per width you get. Possible values are numbers. Negative numbers invert the pattern.
5 (default)
2
-2
20
offsetX | offsetY
you can slide the pattern up, down, left and right. Possible values are numbers.
0 | 0 (default)
0 | 15
100 | 5
