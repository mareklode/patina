createPattern : wave

The wave patterns are calculated using the sine function. That gives us smooth lines in different directions.

Example

"patina" : {
    "type"          : "createPattern",
    "patternName"   : "wave"
}

Options / Parameters

direction
we have 'horizontal', 'vertical', 'diagonalUp', 'diagonalDown', 'concentric' and 'rectangles'
period
defines how many 'waves' per width you get. Should this parameter be renamed as well?
offsetX / offsetY
you can slide the pattern up, down, left and right if you provide these values.
