filter : threshold

This filter is like contrast to the max. Everything about a given treshold gets white or 100% brightness and everything below it gets black or zero.

Basic Example

"patina" : {
    "type"          : "filter",
    "patternName"   : "noise_plasma",
    "filter"        : [{ "name": "threshold" }]
}

Options / Parameters

The threshold filter takes one parameter:

threshold
possible values are numbers numbers between 0 and 1 (ToDo: use everywhere the same range. 0-255, 0-1, or 0-99/100):
0.5 (default)
0.1
0.85
